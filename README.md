# ğŸ“– Blog App <a name="about-project"></a>

The Blog app is a comprehensive web-based platform designed to provide a user-friendly interface for reading and engaging with a wide range of articles and posts. This fully functional website offers a variety of interactive features, including the ability to leave comments and express support for posts through likes, making it an ideal solution for bloggers and content creators looking to engage with their readers in a meaningful way.

## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
- laptop
- git
- Code Editor.
- PostgreSQL.
- Ruby On Rails.

### Setup

Clone this repository to your desired folder:

sh
  cd my-folder
  git clone https://github.com/nilab-khwaja/blog-app.git
